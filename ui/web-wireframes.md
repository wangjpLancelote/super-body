# Web UI Wireframes

## 概览
本文档定义了Web应用的线框图，采用响应式设计原则，支持桌面端和平板端。Web端将提供比移动端更丰富的功能和交互体验。

## 交互设计

### 桌面端特有功能
- **快捷键支持**: Ctrl/Cmd + 快捷操作
- **拖拽上传**: 直接拖拽文件到页面
- **右键菜单**: 上下文相关操作
- **标签页**: 同时打开多个页面
- **键盘导航**: Tab键遍历，Esc键取消

### 响应式设计
- **桌面端** (≥1024px): 完整功能，侧边栏导航
- **平板端** (768-1024px): 适配触摸，简化导航
- **移动端** (<768px): 单列布局，底部导航（已有）

### 状态管理
- **实时更新**: WebSocket连接，实时数据同步
- **离线支持**: Service Worker，本地缓存
- **数据同步**: 跨设备自动同步状态

## 视觉设计

### 配色方案
- **主色调**: #8CD98C (健康绿)
- **背景色**: #FFFFFF (桌面端) / #0B0F14 (移动端深色)
- **文字色**: #1F2937 (深灰) / #F5F7FA (浅灰白)
- **强调色**: #3B82F6 (蓝色) / #F59E0B (橙色)

### 字体系统
- **标题**: 24px - 32px, font-weight: 700
- **正文**: 14px - 16px, font-weight: 400
- **辅助**: 12px - 14px, font-weight: 500

### 间距系统
- **微间距**: 4px, 8px
- **小间距**: 12px, 16px
- **中间距**: 24px, 32px
- **大间距**: 48px, 64px
